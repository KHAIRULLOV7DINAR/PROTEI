Для запуска программы необходимо:

Cоздать папку build на уровне с CMakeLists.txt;
Перейти в нее и выполнить конфигурацию и генерацию файлов сборки: cmake ..;
Собрать проект: cmake --build .;
Вызывать выполнение программы с необходимыми флагами: ./HOMETASK_1 -a 127.0.0.1 -p 5555 -r Client -i 0 -L mylib

Созданное CLI-меню обеспечивает выполнение следующих команд:
name - ввод имени программы;
type - ввод типа значений вектора;
vector - ввод 4-мерного вектора введенного типа;
console - вывод значений, полученных через флаги запуска программы;
menu - вывод значений, полученных из CLI-меню;
exit - выход из CLI-меню.

Некоторые уточнения при выполнения работы:
1. Считалось, что значения, передаваемые в строке консоли при вызове программы, - опциональны. В случае если некоторые из них не указаны, то они заполняются значениями по умолчанию (127.0.0.1, 5555, Client, 0, mylib);
2. Октеты адреса ip ограничены [0; 255], порт ограничен [0; 65535]. Неизвестно, чем ограничен i, поэтому это значение считывалось как обычный int;
3. При двойном указании флага, некорректных значениях -a/-p/-i, указании флага без значения, указания неизвестного из ТЗ флага - программа выдает пользователю ошибку и останавливает выполнение, так как необходим повторный запуск программы с правильными значениями;
4. Считалось, что при повторном вызове команды в текстовом интерфейсе меню мы просто перезаписываем ранее считанное значение;
5. Если не указан тип вектора, то нельзя его ввести;
6. Вектор может иметь три типа значений внутри: int, float, double;
7. При указании нового типа, старый вектор удаляется, при повторном указании того же типа, вектор сохраняет значения;
