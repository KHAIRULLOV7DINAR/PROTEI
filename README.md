# PROTEI
Репозиторий для домашних заданий IT-школы ПРОТЕЙ.

Домашнее задание 1.



Уточнения, которые соблюдались при выполнении домашнего задания:
1. Считается, что значения, передаваемые в строке консоли при вызове программы, - опциональны, в случае если некоторые из них не указаны, то они заполняются значениями по умолчанию (127.0.0.1, 5555, Clien, 0, mylib);
2. Считается, что при повторном вызове команды в текстовом интерфейсе меню мы просто перезаписываем ранее считанное значение.
